# ArenaQA UI 重新设计计划

## 目标
根据提供的截图重新设计 UI，打造更精致的深色主题界面。

## 当前状态分析
- 三栏布局：左(输入) / 中(回答) / 右(摘要) ✅
- 深色模式支持 ✅
- 模型选择渐变色 ✅
- 流式回答显示 ✅
- 评分/融合面板 ✅

## 目标差异分析（基于截图观察）
1. **背景层次** — 需要更精细的深色背景层次（#0f1117 → #1a1d2e → #252836）
2. **卡片边框** — 增加微妙的边框强调（rgba(255,255,255,0.06)）
3. **阴影系统** — 增强深色模式下的阴影层次
4. **导航栏优化** — 简化品牌标识，突出功能入口
5. **模型卡片** — 优化渐变效果和选中态视觉反馈
6. **评分卡片** — 增强评分条的视觉层次
7. **响应式优化** — 确保小屏幕下的可用性

## 实现步骤

### 步骤 1: 更新设计令牌（globals.css）
- 新增深色模式专用色板
- 优化阴影系统（深色模式专用）
- 调整间距和圆角细节
- 增加边框颜色层次

### 步骤 2: 优化导航栏（NavBar.tsx）
- 简化品牌区域
- 增强深色模式视觉效果
- 优化响应式布局

### 步骤 3: 重设计输入面板（InputPanel）
- 优化模型选择卡片样式
- 增强选中态渐变效果
- 改进输入框视觉反馈
- 优化进度条动画

### 步骤 4: 优化回答列（AnswerColumn）
- 增强列头视觉层次
- 优化流式光标动画
- 改进代码块样式（深色模式）

### 步骤 5: 重设计结果面板（SidePanel）
- 优化评分卡片布局
- 增强融合区域视觉效果
- 改进空状态和加载状态

### 步骤 6: 细节打磨
- 微调动画过渡效果
- 优化滚动条样式
- 增强 focus 状态可见性
- 确保无障碍访问

## 预期效果
- 更精致的深色主题
- 更清晰的视觉层次
- 更流畅的交互体验
- 更专业的整体观感

## 验证方式
- 运行 `npm run build` 确保构建通过
- 运行 `npm run dev` 实时预览效果
- 测试深色/浅色模式切换
- 验证响应式布局
